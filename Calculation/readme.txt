程序的运行流程：
    整个流程使用 "启动服务器 -->上传文件 --> 处理文件 --> 处理结果产生的图片显示到前端" 的顺序展开
    1、启动服务器，本机作为服务器 在这个项目中运行下面的 app.py 就可以启动服务器
        app.py 中可以配置 web 项目中有什么 html 界面等
    2、到达 template/光谱仪定标.html 界面，进行文件的上传
        上传的文件为 Standard_Spectral_Data_1114 文件夹下面所有的文件
    3、在 "光谱仪定标.html" 中点击"光谱仪定标"按钮，main.py 和光谱仪定标按钮绑定，开始下面第4步的数据处理
    4、在Calculation/main.py 进行后端的文件处理
        main.py 运行后产生的   Light_*_.conf 文件放置在了 out 文件夹中（out 文件夹不用关注~）
        main.py 运行后产生一系列的 .png 文件，需要展示在前端（按钮点击完成，自动绘制并且展现到前端中）
        展示到前端之后，这个图片需要选择路径，保存在本地
